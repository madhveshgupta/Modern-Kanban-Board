# Modern Kanban Board

A sleek, productive, and responsive Kanban board application designed to streamline task management.

## 🚀 Features

- 🖱️ **Native Drag & Drop:** Smoothly move tasks across "To Do", "In Progress", and "Done" columns.
- 💾 **Persistent Data:** Integrated with browser `localStorage` so your tasks are saved even after a page refresh.
- 🎨 **Modern Dark Theme:** Features a premium glassmorphism navigation bar, custom scrollbars, and dynamic glowing gradients.
- 📊 **Live Task Counters:** Automatically updates the total number of tasks in each column in real-time.

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3 (Custom properties, flexbox, grid, backdrop filters)
- **Behavior:** Vanilla JavaScript (ES6+, DOM Manipulation, LocalStorage API)
